Testing News Education Guide Placement

Open developer tools and run this in console on the stock details page:

// Check if the educational guide is present
const guide = document.querySelector('[class*="GraduationCap"]');
console.log('Educational guide found:', !!guide);

// Find all section headers in the news tab
const newsTab = document.querySelector('[data-state="active"][value="news"]');
if (newsTab) {
    const headers = newsTab.parentElement.querySelectorAll('h2, h3, h4, [class*="CardTitle"]');
    console.log('Section headers in order:');
    headers.forEach((h, i) => console.log(`${i + 1}.`, h.textContent.trim()));
}

// Check the order of components
const components = document.querySelectorAll('[class*="Card"]');
console.log('Found', components.length, 'card components');

Expected Order:

  1. Why News Matters for Investors (Educational Guide)
  2. AI News Intelligence (AI Summary)
  3. Latest News & Events (News Feed)

Troubleshooting Steps:

  1. Hard refresh the page (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows)
  2. Clear browser cache
  3. Check if you're on the News & Events tab
  4. Wait a moment for the page to fully load
